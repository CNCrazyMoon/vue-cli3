## 5.01 澳门一日游 [参考攻略](http://www.mafengwo.cn/i/11074194.html)
1. 准备：身份证，港澳通行证、货币（建议在拱北买，入境大厅门口就有）
2. 出发：争取 `07:00` 到达 `拱北口岸`开始入境
3. 整体路线：拱北口岸 -> 威尼斯人 -> 官也街/圣母街十字公园(2地很近) -> 大三巴 -> 新葡京 -> 拱北口岸
* 拱北口岸 -> 威尼斯人
